# FHE vs. MPC for ML

This repository is the collection of code used in the paper Comprehensive Benchmarking of Secure Computation Technologies for Machine Learning on General Purpose Hardware published in the special issue Cloud Computing: New Network Technology and Information Security of MDPI and can be found under https://www.mdpi.com/2076-3417/16/13/6519. 

## Installation

To use this repository just clone it into the desired folder. In it both subfolders are filled with benchmarks and directories used for result storing. They are managed by uv managed virtual python environments, but it is not strictly necessary to use uv. Both can function without using it. More informations on their benchmarks can be found in the respective subdirectories.
